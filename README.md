Client Interface for LTCRabbit.com API
======================================

Requirements
============

* Python 2.6 or above
* `requests`_ library
* `docopt`_ library
* `prettytable`_ library

# Usage

### Worker List

    ltcrabbit-cli worker list --apikey=<your_api_key>

### User Info

    ltcrabbit-cli user --apikey=<your_api_key>
    
### Pool Info
    
    ltcrabbit-cli pool --apikey=<your_api_key>
